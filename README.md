# fwwb_A09_Rail_transit

#### 一、介绍
**【问题说明】**     
以地铁 ACC(地铁自动售检票系统清分中心简称)系统的用户行程数据、站点 数据为基础，完成基于地铁出行行程大数据的分析建模和算法研究，实现对地铁 的线路级别以及站点级别的客流进行分析和预测。  
**【用户期望】**     
提供友好的用户交互方式，通过输入或者调整模型的各种相关因子，对指定 时间、指定线路或者站点的客流进行预测和预警并且通过图形化的方式直观展 现。
